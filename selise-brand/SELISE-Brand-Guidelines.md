## 2. Logo System

### Primary Logo

The primary logo features a distinctive icon paired with clean, modern typography.
Use it in high-visibility areas: website headers, marketing materials, official documents.

### Unit Logos

Each unit logo retains the core SELISE logomark and typeface. The unit descriptor
appears beneath or beside the main logotype in a standardized format.

### Minimum Size

| Logo Type      | Min Width | Approx. Print Size |
| -------------- | --------- | ------------------ |
| Primary Logo   | 90px      | ~1.25 inches       |
| Secondary Logo | 50px      | ~0.28 inches       |

### Clear Space

Always maintain clear space around the logo. Use on a clear or transparent background
for versatile applications.

### Positioning Rules

| Placement           | Allowed? |
| ------------------- | -------- |
| Top-left            | Yes      |
| Top-right           | Yes      |
| Bottom (centered)   | Yes      |
| Other odd positions | **No**   |

### Logo Do's

- Use on a clean, uncluttered background
- Display in original proportions without distortion
- Use approved color variations (black, white, or primary blue) based on background contrast
- Place prominently for easy recognition

### Logo Don'ts

- No low-contrast or busy backgrounds
- No unapproved color alterations
- No stretching, skewing, or disproportionate resizing
- No shadows, gradients, or outlines applied to the logo
- No unapproved colors or clashing graphic elements
- No colorful or patterned backgrounds

### Background Usage

| Background Type | Logo Version                                                       |
| --------------- | ------------------------------------------------------------------ |
| Light           | Darker text/icons (blue or black version)                          |
| Dark            | Lighter text/icons (white version)                                 |
| Colored         | Ensure sufficient contrast; use semi-transparent overlay if needed |

## 4. Color System

### Primary Palette

| Name        | Hex       | Role                 |
| ----------- | --------- | -------------------- |
| SELISE Blue | `#0066B2` | Primary brand color  |
| Oxford Blue | `#001F35` | Deep accent          |
| Globe Grey  | `#7B7C7F` | Secondary neutral    |
| Eerie Black | `#1B2021` | Text primary (dark)  |
| White       | `#FFFFFF` | Text primary (light) |

### Highlight / Accent Colors

| Name        | Hex       | Suggested Use                            |
| ----------- | --------- | ---------------------------------------- |
| Lavender    | `#BC63D7` | Infographics, graphs, icons, visual KPIs |
| Poly Green  | `#2A4D14` | Body content, labels on lists            |
| Lime Green  | `#7BC950` | Positive outcomes, list data             |
| Crimson     | `#D80032` | Risks, issues, pain points, warnings     |
| Pantone Red | `#EF233C` | Negative headers, problem areas          |
| Mauveine    | `#791E94` | Infographics, graphs                     |

### SELISE Blue Shade Ramp

| Shade | Hex       |
| ----- | --------- |
| 100%  | `#0066B2` |
| 80%   | `#247EC1` |
| 60%   | `#4796D1` |
| 40%   | `#6BAEE0` |
| 20%   | `#7DBAE8` |
| 10%   | `#8EC6F0` |

### Color Ratio Guidelines

**3-Color Palette (6:3:1):**

| Role      | Ratio | Usage                                     |
| --------- | ----- | ----------------------------------------- |
| Primary   | 60%   | Backgrounds, large sections, key branding |
| Secondary | 30%   | Subheadings, graphics, secondary buttons  |
| Accent    | 10%   | CTAs, key details, important elements     |

**4-Color Palette (5:2.5:1.5:1):**

| Role      | Ratio | Usage                               |
| --------- | ----- | ----------------------------------- |
| Primary   | 50%   | Backgrounds, major branding         |
| Secondary | 25%   | UI elements, illustrations, content |
| Tertiary  | 15%   | Borders, icons, subtle text         |
| Accent    | 10%   | Links, alerts, interactive details  |

### Light & Dark Mode Variants

Both the 3-color and 4-color palettes have **Light** and **Dark** variants. Choose based on context:

- **Light mode:** White/light backgrounds dominate; use SELISE Blue and dark text for contrast
- **Dark mode:** Dark backgrounds dominate; use lighter blues and white text for contrast

### Categorical Color Usage

| Category                     | Color        | Use For                                                |
| ---------------------------- | ------------ | ------------------------------------------------------ |
| Text / Info                  | Dark Green   | Body content, labels on lists                          |
| Infographics                 | Purple       | Graphs, icons, visual stats or KPIs                    |
| Warning / Negative Headers   | Bright Red   | Highlighting risks, issues, pain points, problem areas |
| Positive Outcome / List Data | Green shades | Success indicators, positive metrics, list data        |

### Lighter Shades Usage

Lighter shades are used for creating a clean, professional look:

- **Backgrounds** — keep slides/pages minimal and readable
- **Secondary text** — ensuring clarity without distraction
- **Hover & Active States** — subtle visual cues on interactive elements
- **Section Dividers** — break up content without overpowering it
- **Charts & Graphs** — highlight secondary data while maintaining focus on primary information

### Color Usage Rules

- Ensure proper contrast between text and background for readability
- Use primary colors (`#0066B2`, `#7B7C7F`) to maintain consistent identity
- Apply secondary/highlight colors sparingly to support the primary palette
- Use `#FFFFFF` and `#1B2021` for text
- Maintain contrast ratios for accessibility (white text over dark, dark text over light)
- Use darker shades for primary actions/headings, lighter ones for backgrounds/layering
- Don't overload pages with too many colors; use accents strategically

## 5. Gradients

| Name                 | From      | To        | Use Cases                                      |
| -------------------- | --------- | --------- | ---------------------------------------------- |
| Light Gradient       | `#F9F9F9` | `#EDEDED` | Backgrounds, subtle overlays, section dividers |
| Mid Gradient (Sky)   | `#5BAEFF` | `#0073C6` | Web banners, accent components, CTAs           |
| Dark Gradient (Deep) | `#0067A3` | `#002D72` | Hero sections, footers, impactful highlights   |

### Gradient Rules

- Apply consistently with appropriate contrast for text and icons
- Maximum one gradient per component
- Gradients should complement, not compete with, imagery or other brand elements

## 6. Branding Applications

### Do's

- Apply branding consistently across all mediums (digital, print, events)
- Test designs in various formats to ensure adaptability and quality

### Don'ts

- Do not deviate from the approved guidelines for any medium
- Do not compromise brand integrity by using low-quality materials or incorrect logo placements

## 7. Photography & Imagery

### Do's

- Use sharp, well-lit, properly exposed images
- Align imagery with brand message and target audience
- Maintain consistent styling across marketing materials

### Don'ts

- No generic or inauthentic stock images
- No low-resolution or poorly lit images
- No excessive editing or filters that distort the natural look

### Background Images

**Where to use:** Corporate presentations, web/digital platforms, marketing materials, event branding

**Prohibited:**
- Overly complex or distracting designs
- Stock images with no connection to SELISE
- Low-resolution or pixelated images

## 8. Brand Elements & Icons

### Brand Elements

- Primarily rendered in **monochrome (grayscale)** for modern, professional appearance
- Alternative: white (on dark backgrounds) or brand blue for visibility
- Maintain visual harmony, proper spacing, alignment, and size proportions

### Icon Sources

Only use icons from these two sources:

1. **Flaticons** — copy/paste into presentations; use black, grey, or white colors
2. **unDraw.co** — download and insert images; match presentation colors

### Icon Style Guidelines

| Rule              | Detail                                                      |
| ----------------- | ----------------------------------------------------------- |
| Style Consistency | Use outlined or flat-style icons only; never mix styles     |
| White Icons       | Use over dark or gradient blue backgrounds (high contrast)  |
| Black Icons       | Use on white or light backgrounds (clean visual hierarchy)  |
| Customization     | Recolor to brand palette (blue tones or grayscale)          |
| Effects           | No distortion, stretching, shadow, or glow effects          |
| Sizing            | Consistent sizing and spacing; align with accompanying text |
| Purpose           | Icons enhance content, not distract; use sparingly          |

## 9. Co-Branding

### Rules

- SELISE logo must remain in original form
- Equal visual weight and spacing between logos
- Use neutral backgrounds and consistent typography
- Client logo is placed **before** the SELISE logo
- Strive for harmony — no brand should overpower the other
- All co-branded materials must be pre-approved

## 10. Social Media

### General Rules

- Use SELISE primary colors for backgrounds, text overlays, and icons
- Visually align posts with brand tone and message
- Balance promotional content with value-driven posts

### Platform Dimensions

| Platform / Format         | Size / Ratio      |
| ------------------------- | ----------------- |
| Instagram Post            | 1080 x 1080 (1:1) |
| Instagram Stories / Reels | 9:16 (Vertical)   |
| Facebook / LinkedIn Post  | 1.91:1 or 1:1     |
| Company Logo              | 400 x 400         |
| Company Cover             | 646 x 220         |
| Update Post               | 1080 x 1080       |

### Carousel Posts

- Cohesive visual elements and consistent typography
- High-quality imagery with primary colors for emphasis
- Clear narrative flow across slides to encourage swiping

### Blog Posts

- Primary and secondary colors for headers, subheadings, callout elements
- High-quality imagery aligned with blog content
- Professional tone, engaging and informative
- Consistent typography and spacing for readability
