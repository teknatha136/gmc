Setup

Before you start

You'll run every problem in VS Code with Node.js. No browser, no HTML — just JavaScript and the terminal.

  1. Create a new folder called js-practice and open it in VS Code.
  2. For each problem, create a new file (e.g. 1-1-even-or-odd.js). Keeping one file per problem makes it easy to come back later.
  3. Open the integrated terminal (View → Terminal or Ctrl + `).
  4. Run your file with node 1-1-even-or-odd.js.
  5. Use console.log(...) to print. Always test your function with a few inputs before moving on.
  6. Stuck for more than 10 minutes? Raise a hand. Don't burn a whole block on one problem.

1.3 Leap year && · ||

Write isLeapYear(year) that returns true for leap years (years with an extra day, like Feb 29). Check these rules in order:

1. If divisible by 400 → leap year
2. Else if divisible by 100 → not a leap year
3. Else if divisible by 4 → leap year
4. Else → not a leap year

1-3-leap-year.js
// Examples

isLeapYear(2023); // false (not ÷4)

4.8 Sort numbers ascending .sort() + comparator

Write sortAscending(nums) that returns an array of numbers sorted from smallest to largest.

Heads-up: the plain .sort() with no arguments sorts numbers as text — so [10, 2, 30] becomes [10, 2, 30] (treated as "10", "2", "30"), not [2, 10, 30]. You'll need to pass a comparator function.

4-8-sort-ascending.js
sortAscending([10, 2, 5, 1, 30]); // [1, 2, 5, 10, 30]
sortAscending([-3, -10, -1]); // [-10, -3, -1]
sortAscending([7]); // [7]
