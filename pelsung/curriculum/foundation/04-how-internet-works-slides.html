06 IP Addresses
IP Addresses

Every Device Has an Address

🏠 Think of a Postal Address

  • Your house has an address so people can find it
  • Your device has an IP address so the internet can find it
  • IP = Internet Protocol
  • Every device on the internet needs one

🔢 What an IP Looks Like

192 . 168 . 1 . 1
  • 4 numbers, each 0–255
  • IPv4: ~4 billion addresses (running out!)
  • IPv6: new format, essentially unlimited

selisegroup.com has the IP address 185.34.33.33 — that's how your browser really finds it

10 HTTP & HTTPS
Protocols

HTTP — The Language of the Web

🔓 HTTP

  • HyperText Transfer Protocol
  • The rules for how browsers and servers talk
  • Data travels in plain text
  • Anyone on the network can read it
  • URL starts with http://

🔒 HTTPS — HTTP + Security

  • S = Secure (uses TLS encryption)
  • Data is encrypted end-to-end
  • Padlock icon in the browser bar
  • Required for passwords, payments, forms
  • URL starts with https://

🛡️ Always check for HTTPS before entering a password or card number. If the padlock is missing — don't type it.

11 Status Codes
HTTP Status Codes

The Server's Reply Code

Every HTTP response starts with a 3-digit number

200
OK
Request succeeded. Here's your data.
301
Moved
Page has a new permanent address.
404
Not Found
Page doesn't exist on this server.
403
Forbidden
You don't have permission.
500
Server Error
Something broke on the server.

2xx = success · 3xx = redirect · 4xx = your fault · 5xx = server's fault

12 The Full Journey
Step by Step

You Type selisegroup.com

  • ① Browser checks its cache — has it visited selisegroup.com recently?
  • ② DNS lookup — asks DNS resolver: "what's the IP for selisegroup.com?" → gets 185.34.33.33
  • ③ Browser opens a TCP connection to that IP address (port 443 for HTTPS)
  • ④ TLS handshake — browser and server agree on encryption keys
  • ⑤ Browser sends: GET / HTTP/1.1 Host: selisegroup.com
  • ⑥ Server processes the request, finds the homepage files
  • ⑦ Server responds: 200 OK + sends back HTML
  • ⑧ Browser parses HTML, fetches CSS/JS/images, renders the page ✅
