05 HTML
HyperText Markup Language

HTML is the skeleton of a page

You "mark up" your content with tags that say what each piece is: a heading, a paragraph, a list, a link, an image

<h1>
Opening
tag
Hello, Gelephu!
Content
</h1>
Closing
tag

Tags come in pairs. The browser reads this and displays "Hello, Gelephu!" as a big heading.

06 Page Skeleton
The Skeleton

Every HTML page starts the same way

index.html
<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello, Gelephu!</h1>
    <p>Welcome to my first web page.</p>
  </body>
</html>

<html>

Wraps everything. The root of the page.

<head>

Info the browser needs: title, CSS links, fonts. Not shown on the page itself.

<body>

Everything you actually see on the page goes here.

07 Common Tags
Your Starter Vocabulary

Nine tags cover 80% of most pages

<h1> … <h6>

Headings. h1 is biggest, h6 is smallest.

<p>

A paragraph of text.

<a href="…">

A link to another page or site.

<img src="…">

An image. Needs a src (source) attribute.

<ul> + <li>

Unordered (bulleted) list. Each <li> is one item.

<ol> + <li>

Ordered (numbered) list.

<div>

A generic box. Used to group stuff together.

<span>

A generic inline piece. For styling part of a sentence.

<button>

A clickable button.

Cascading Style Sheets

If HTML is the skeleton,
CSS is the skin & clothes

Colours, fonts, spacing, layout, animation — all controlled by CSS

style.css
/* "h1 tags should look like this" */
h1 {

10 Rule Anatomy
The Three Pieces of a CSS Rule

Selector · Property · Value

🎯

Selector

Which elements do you want to style?
h1 — all h1 tags
.btn — class "btn"
#hero — id "hero"

⚙️

Property

What aspect do you want to change?
color, background, font-size, margin, padding, width, display…

✨

Value

The setting for that property.
#0066B2, 16px, center

11 Box Model
The Most Important CSS Concept

Every element is a box

Content inside, padding around the content, border around the padding, margin outside everything

margin
border
padding
CONTENT

Content

The actual text or image inside.

Padding

Space inside the border. Breathing room around content.

Border

The edge. Can be visible or invisible.

Margin

Space outside. Pushes neighbours away.

12 Colors & Fonts
The Two Most-Used Properties

Colors & Fonts

🎨 Colors

Four ways to say the same blue:

blue — by name
#0066B2 — hex
rgb(0, 102, 178) — RGB
hsl(205, 100%, 35%) — HSL

Hex is most common. SELISE blue is #0066B2.

✏️ Fonts

Font-family + font-size + weight:

font-family: 'Open Sans', sans-serif;
font-size: 16px;
font-weight: 700;
line-height: 1.6;
text-align: center;

Weight 400 = normal, 700 = bold. You can load fonts from Google Fonts for free.

13 Flexbox
Layout

Flexbox — line things up in a row or column

Add display: flex to a container — its children become flex items and line up side-by-side

A
B
C
style.css
.container {
  display: flex;
  gap: 1rem;   /* space between items */
  justify-content: center;  /* horizontal alignment */
}
14 Connecting Them
Linking

How HTML finds your CSS

One line in the <head> tells the browser to load your stylesheet

index.html — the head section
<head>
  <title>My Page</title>
  <link rel="stylesheet" href="style.css">
</head>

Two files: index.html (structure) and style.css (look). The link tag glues them together.

18 Exercise
Follow Along · Ask When Stuck

Build an "About Me" page

  • 1
    In VS Code, create a new folder about-me and two files inside: index.html and style.css.
  • 2
    In index.html, type the page skeleton (<html>, <head>, <body>) and add a link to style.css.
  • 3
    In the body: a <h1> with your name, a <p> about yourself, a <ul> with 3 interests or skills, and a <img> of anything you like (photo, flag, logo).
  • 4
    Open index.html in your browser. It should look plain — just black text. Use the Live Server extension to auto-reload on save.
  • 5
    In style.css, change the page's background colour, the font, the h1 colour, and the bullet list spacing. Save and watch the browser update.
  • 6
    Stretch: wrap your content in a <div class="card">, give the card a width, padding, a background, and centre it on the page with flexbox.
